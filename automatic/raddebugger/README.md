## RAD Debugger

The RAD Debugger is a native, user-mode, multi-process, graphical debugger. This package automates the installation of the debugger toolchain, including:

- raddbg.exe: The core graphical debugger.

- radbin.exe: The RAD Debug utility for RDI (Relative Debug Info) files.

- radlink.exe: The RAD Linker utility.
